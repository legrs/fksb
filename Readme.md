// control
姿勢制御のため，ドローン用のブラシレスモータに座金を取り付けてリアクションホイールとしました．
11.1Vのリポバッテリを電源としています．
制御はPD制御で行っています．
リアクションホイールの慣性モーメントと機体の主慣性モーメントはFreeCADとPythonで算出し，
缶サットに搭載された9軸センサからの角速度と姿勢の値を得られます．
Eularの運動方程式に基づいて必要トルクを求め，ESCに送るPWMを決定します．

今回はゲームエンジンの物理演算機能を制御システムの実証に使うことで，制作時間を短縮しました．



// app
我々の開発した地上局アプリでは，缶サット本体からのテレメトリの受信と，地上からのコマンド送信が可能です．
受信にもTweliteを使用していて，Tweliteとのserial通信にはc++のlibftdiを使用しています．

地方大会ではc++のOpenGLを用いたインタフェースでしたが，利便性や安定性の向上のため
今回はC#とc++の両方を使えるGodotというオープンソースのゲームエンジンを用いたGUIアプリを作成しました．

このアプリでは，リアルタイムで姿勢を3D表示したり，グラフを描画することができます．
また，キーボードからの入力でコマンドを送信し，機体の動作モードを切り替えることができます．

缶サット本体とUSB接続すると，メモリチップに記録されたデータの抽出にも使用できます．
